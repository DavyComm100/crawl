This article is for Dropbox users on an old release of Dropbox Paper.

To find out if you’re using the old release of Dropbox Paper:

  1. Sign in to dropbox.com.
  2. Click Create.
  3. Hover over Document.

If you don’t see Dropbox Paper in the list of document types, then you’re using the old release.

If you do see Dropbox Paper in the list of document types, then you’re using the new 2020 release of Dropbox Paper.

If you’re using the 2020 release of Dropbox Paper, your Paper docs act like regular Dropbox files, and you can change privacy settings for your Paper docs the same way you would change privacy settings on regular files in Dropbox. Learn how to change privacy setting on Dropbox files and folders.

If you create a private doc that’s not inside of a folder, you’re the only person that will be able to search for and edit it. Once you begin sharing the doc with others, you can choose who can open your doc and whether they can comment or edit.  If you create a doc within a folder, all members of that folder can open, search for, and edit your doc.

Not using Dropbox yet? See how Dropbox makes managing file permissions easy.

Change sharing permissions on a doc

  1. Open the Paper doc you want to share. 
  2. Click the blue Invite button.
  3. Click Settings at the bottom right.
  4. Choose a sharing option under both Who can open this doc? and What can they do?.
  5. Click Done.

Change sharing permissions on a doc for an individual

  1. Open the Paper doc.
  2. Click the blue Invite button.
  3. Next to their name, choose Can edit, Can view, or Remove.
  • You can't remove yourself from a doc, but you can unfollow it
  • If you’re on a team and link access is enabled, the only way to remove a user from a doc is to set the doc’s permissions to Invite-only

Change sharing permissions on a Paper folder

  1. Open the folder you want to share. 
  2. Click the blue Share button.
  3. Type email or members names next to To.
  4. Click their name.
  5. Click Invite.

Notes:

  • All folders within a shared folder are automatically shared as well
  • If you're on a Dropbox Business team your Paper folders will be visible to everyone by default

How to remove a user from a doc

To remove someone from the doc, you must either have created the doc or have edit access. 

To remove a user from a doc:

  1. Click the blue Invite or Share button.
  2. Click Can edit (or Can view) next to the relevant user. 
  3. Click Remove from the dropdown menu that appears.

Notes: 

  • You can't remove yourself from a doc, but you can unfollow it
  • If you’re on a team and link access is enabled, the only way to remove a user from a doc is to set the doc’s permissions to Invite-only

How to see who your Paper folder is shared with

To see who a Paper folder is shared with:

  1. Open the folder.
  2. Click the blue Share button.

You’ll see a list of people you’re sharing the folder with.
