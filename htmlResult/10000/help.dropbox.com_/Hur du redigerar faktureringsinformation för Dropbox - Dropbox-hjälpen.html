Följ instruktionerna för din kontotyp om du vill ändra din faktureringsinformation för Dropbox, inklusive kreditkortsnummer.

Om du vill säga upp kontot, läs om hur du säger upp din Dropbox-prenumeration. 

Obs! Om du betalar med manuell faktura och vill ändra faktureringsinformationen för Dropbox måste du kontakta Dropbox support eller den kontoansvarige.

Dropbox Plus, Family eller Professional.

Så här ändrar du din faktureringsinformation:

  1. Logga in på dropbox.com.
  2. Klicka på din avatar (profilbild eller initialer) längst upp till höger. 
  3. Klicka på Inställningar.
  4. Klicka på Fakturering.
  5. Klicka på Uppdatera bredvid Betalningsmetod.
    • Obs! Om du köpte din prenumeration på en mobil enhet måste du uppdatera din faktureringsinformation genom Google Play eller iTunes istället.
  6. Lägg till ett nytt kreditkort eller koppla ett PayPal-konto. 
    • Obs! Kunder i Indien har möjlighet att använda UPI som betalningsmetod i kassan.
  7. Klicka på Spara ändringar.

Obs! Endast den som ansvarar för Family-kontot för en Dropbox Family-plan kan ändra faktureringsinformationen.

Dropbox-administratörer på teamkonton

Vissa typer av administratörer kan ändra faktureringsinformationen för deras teams Dropbox Standard- eller Advanced-konto. 

Så här ändrar du din faktureringsinformation:

  1. Logga in på dropbox.com med autentiseringsuppgifterna som administratör.
  2. Klicka på Adminkonsolen på sidomenyn till vänster.
  3. Klicka på Fakturering.
  4. Klicka på Ändra faktureringsinformation bredvid Betalningsmetod.
  5. Lägg till ett nytt kreditkort eller länka till ett PayPal-konto. 
    • Obs! Endast administratören som anger faktureringsinformationen kan se det angivna kreditkortsnumret. Det här fältet visas som tomt för alla andra administratörer som försöker att se det.
  6. Klicka på Spara ändringar.

Obs! Kunder i Indien har möjlighet att använda UPI som betalningsmetod i kassan.

Varför kan jag inte hitta min Dropbox-faktureringsinformation?

Kontrollera följande om du fortfarande inte hittar faktureringsinformationen:

  • Om du köpte din Dropbox-prenumeration på en mobil enhet måste du uppdatera din faktureringsinformation genom Google Play eller iTunes istället.
  • Om du är en teammedlem med ett Dropbox Standard-, Advanced- eller Enterprise-konto kan du inte se din faktureringsinformation. Endast vissa typer av administratörer kan se och ändra faktureringsinformationen.
  • Om du är med i ett Dropbox Enterprise- eller Education-team eller om du köpte via en återförsäljare, kan du inte se faktureringsinformation i ditt konto.
  • Du kanske oavsiktligt har loggat in på ett kostnadsfritt Dropbox-konto som du inte visste att du hade. Använd vårt sökverktyg för kreditkortsdebitering för att hitta kontot som är associerat med en debitering på kreditkortet.

Kontakta supporten om inget av ovanstående gäller och du fortfarande inte kan hitta din faktureringsinformation.
