Você pode criar um link para um arquivo ou pasta na sua conta do Dropbox e compartilhá‑lo com outras pessoas. Ao compartilhar um arquivo ou pasta através de um link, você pode optar por dar acesso de edição ou somente de visualização às pessoas que tiverem esse link. Saiba como gerenciar suas configurações-padrão de compartilhamento.

Se você compartilhar o link por e‑mail, WhatsApp, Twitter, mensagem instantânea ou outro aplicativo de terceiros, o destinatário verá o nome do arquivo e uma miniatura do conteúdo. Quando o destinatário clicar no link, o arquivo será aberto como uma visualização prévia no dropbox.com e os destinatários poderão deixar comentários no arquivo.

Observações: 

  • Arquivos do Microsoft Office e Google Docs, Sheets e Slides permitem acesso de edição por padrão, mas você pode alterar o acesso para somente visualização ao criar o link.
  • Os clientes dos planos Dropbox Professional, Standard, Advanced e Enterprise podem configurar permissões adicionais para os links compartilhados, como senhas e prazos de validade.
  • Não é possível criar um link compartilhado para uma pasta restrita.

Ainda não está usando o Dropbox? Veja como o Dropbox ajuda você a compartilhar pastas com facilidade

Criar e compartilhar um link do Dropbox com acesso de edição

Se você tiver uma conta do Dropbox Basic, Professional ou Plus, pode dar a outros usuários do Dropbox acesso de edição aos seus arquivos ou pastas do Dropbox através de um link. Se você compartilhar o link com uma pessoa que não tiver uma conta do Dropbox, ela só terá acesso à visualização prévia do arquivo ou pasta. 

Atenção: Se você conceder acesso a um arquivo ou pasta a outro usuário do Dropbox, ele poderá criar seu próprio link para compartilhar esse conteúdo com outras pessoas. Se você for administrador de uma equipe do Dropbox, pode controlar as configurações de link para todos. Como gerenciar as configurações de compartilhamento de links para a sua equipe.

No dropbox.com

  1. Acesse o site dropbox.com.
  2. Passe o mouse sobre o arquivo ou pasta que deseja compartilhar e clique no ícone de compartilhamento (retângulo com seta para cima).
  3. No menu suspenso, selecione a opção Pode editar.
    • Caso um link não tenha sido criado, clique em Criar.
    • Se um link tiver sido criado, clique em Copiar link.
  4. O link será copiado para a área de transferência. Você poderá então colá‑lo em um e‑mail, mensagem ou qualquer outro lugar onde as pessoas possam acessá‑lo.

Observação: Se o destinatário do link for membro de uma conta de equipe do Dropbox, a possibilidade de editar o conteúdo dependerá das permissões de sua equipe.

Criar e compartilhar um link do Dropbox com acesso somente de visualização

Se alguém receber um link ou arquivo compartilhado com permissões somente de visualização, não poderá editar o arquivo original.

No dropbox.com

  1. Acesse a sua conta no site dropbox.com.
  2. Passe o mouse sobre o arquivo ou pasta que deseja compartilhar e clique no ícone de compartilhamento (retângulo com seta para cima).
  3. No menu suspenso, selecione Pode visualizar.
    • Caso um link não tenha sido criado, clique em Criar.
    • Se um link tiver sido criado, clique em Copiar link.
  4. O link será copiado para a área de transferência. Você poderá então colá‑lo em um e‑mail, mensagem ou qualquer outro lugar onde as pessoas possam acessá‑lo.

Em seu computador

  1. Abra a pasta do Dropbox no Explorador de Arquivos (Windows) ou no Finder (Mac).
  2. Clique com o botão direito do mouse ou pressione Command e clique no arquivo ou pasta que deseja compartilhar.
  3. Clique em Compartilhar….
    • Caso um link não tenha sido criado, clique em Criar link.
    • Caso um link tenha sido criado, clique em Copiar link ao lado de Pode visualizar.
  4. O link será copiado para a área de transferência. Você poderá então colá‑lo em um e‑mail, mensagem ou qualquer outro lugar onde as pessoas possam acessá‑lo.

Gerenciar links com acesso de edição ou somente visualização

Você pode alterar um link para um arquivo ou pasta depois de compartilhá‑lo. Veja como gerenciar as permissões de compartilhamento do Dropbox.

Existem algumas exceções:

  • Se você remover o acesso de alguém e essa pessoa ainda tiver um link para um arquivo ou pasta, ela poderá recuperar o acesso. Para evitar que o acesso seja recuperado, exclua o link. Veja como remover um link para um arquivo ou pasta.
  • Se uma pessoa com quem você compartilhou um link para uma pasta adicionar essa pasta à conta do Dropbox dela, as permissões que você definiu para o link não serão mais válidas. Saiba mais sobre como adicionar pastas compartilhadas.
  • Se você conceder a alguém acesso de edição a um arquivo ou pasta, ou qualquer acesso a um arquivo baseado na web, como Paper ou Google Docs, essa pessoa poderá criar seu próprio link compartilhado. Você não poderá gerenciar esses links compartilhados; portanto, compartilhe arquivos e pastas somente com pessoas em quem você confia.

Personalizar links e configurações

Os clientes do Dropbox Professional, Standard, Advanced e Enterprise podem alterar as configurações dos links e personalizar os links compartilhados com identidade visual profissional.
