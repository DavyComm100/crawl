Add-On Alat Kreatif Dropbox digantikan dengan Add-On Dropbox Replay, yang dilengkapi berbagai fitur tambahan. Akun tim Dropbox yang telah membeli Add-On Alat Kreatif Dropbox tetap dapat menggunakannya hingga akhir periode penagihan. Pada saat perpanjangan, Add-On Alat Kreatif akan digantikan dengan Add-On Replay.

Ketahui selanjutnya tentang Add-On Dropbox Replay.

Add-On Alat Kreatif untuk paket tim Dropbox dirancang untuk menyederhanakan pekerjaan dengan file media.

Transfer file besar

Fitur transfer file besar meningkatkan batas ukuran Dropbox Transfer, dari 2 GB atau 100 GB, menjadi 250 GB.

Semua pelanggan paket Dropbox Advanced, Enterprise, atau Education, termasuk yang belum membeli Add-On Alat Kreatif, juga dapat menyesuaikan transfer dengan tanggal kedaluwarsa, kata sandi, dan logo atau latar belakang, serta mengetahui orang yang telah melihat atau mengunduh file yang ditransfer.

Ketahui lebih lanjut tentang Dropbox Transfer.

Pratinjau file besar

Fitur pratinjau file besar meningkatkan batas ukuran file audio dan video yang dipratinjau di Dropbox, menjadi 150 GB.

Dengan pratinjau file besar, Anda dapat melihat dan mengomentari file besar tanpa harus mengunduhnya, atau membukanya di aplikasi asli.

Ketahui lebih lanjut tentang mempratinjau file.

Komentar untuk frame tertentu secara akurat

Dengan komentar untuk frame secara akurat, Anda dapat melihat dan mengomentari jenis file video yang umum digunakan, termasuk QuickTime, MPEG-4, dan MXF, hingga dalam level frame.

Ketahui lebih lanjut tentang mengomentari file audio dan video.
