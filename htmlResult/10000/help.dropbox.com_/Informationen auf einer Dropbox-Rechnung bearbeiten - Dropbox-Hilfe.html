Wenn Sie den Rechnungsnamen, die Rechnungsadresse oder die Umsatzsteuer-Identifikationsnummer (nur bei zukünftigen Rechnungen) auf Ihrer Dropbox-Rechnung aktualisieren möchten, folgen Sie der nachstehenden Anleitung für Ihr Abo.

Dropbox Professional 

So bearbeiten Sie die Zahlungsdaten früherer oder zukünftiger Rechnungen für Ihr Dropbox Professional-Konto:

  1. Melden Sie sich bei dropbox.com an.
  2. Klicken Sie rechts oben in der Ecke auf den grauen Kreis oder Ihr Profilbild. 
  3. Klicken Sie auf Einstellungen.
  4. Klicken Sie auf den Tab Abrechnung.
  5. So bearbeiten Sie zukünftige Rechnungen: Klicken Sie neben Rechnungsangaben entweder auf Aktualisieren oder auf Rechungsangaben hinzufügen.
  6. So bearbeiten Sie frühere Rechnungen: Klicken Sie unter Abrechnungsverlauf neben der Rechnung, die Sie ändern möchten, auf Bearbeiten.
  7. Klicken Sie auf Änderungen speichern.

Dropbox Business 

Bestimmte Typen von Dropbox Business-Administratoren können die Zahlungsdaten für frühere oder zukünftige Rechnungen bearbeiten oder anpassen. Und zwar so:

  1. Melden Sie sich als Admin bei dropbox.com an.
  2. Klicken Sie links auf Verwaltungskonsole. 
  3. Klicken Sie auf den Tab Abrechnung.
  4. Klicken Sie auf Abrechnungsverlauf.
  5. So bearbeiten Sie eine frühere Rechnung: Klicken Sie neben der Rechnung, die Sie ändern möchten, auf Bearbeiten.
  6. So bearbeiten Sie eine zukünftige Rechnung: Klicken Sie rechts auf die blaue Schaltfläche Angaben bearbeiten.
  7. Dann können Sie den Namen des Rechnungsempfängers, die Rechnungsadresse, den Wohnort, das Bundesland und ggf. die Umsatzsteuernummer ändern.
  8. Klicken Sie auf Änderungen speichern.

Hinweise:

  • Wenn Sie eine Umsatzsteuer-Identifikationsnummer bearbeiten, gelten die steuerlichen Änderungen nur für zukünftige Rechnungen.
  • Wenn Ihr Team die Zahlung auf Rechnung gewählt hat, wenden Sie sich unter invoices@billing.dropbox.com an uns, um die Angaben auf Ihrer Rechnung zu ändern.

Dropbox Plus

Wenn Sie ein Dropbox Plus-Abo haben, können Sie den Namen Ihres Unternehmens ändern. Damit der Name Ihres Unternehmens auf zukünftigen Rechnungen erscheint, muss er mit dem Namen Ihres Kontos übereinstimmen. Weitere Informationen zum Ändern des Kontonamens.
