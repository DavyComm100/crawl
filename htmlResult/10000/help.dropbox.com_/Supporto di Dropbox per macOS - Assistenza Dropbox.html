Dropbox sta continuando ad ampliare l’implementazione del supporto completo per macOS 12.5 e versioni successive. L’ultima versione di Dropbox per macOS usa l’API Provider File aggiornata di Apple per risolvere problemi relativi all’apertura dei file Solo online da parte di applicazioni di terze parti. L’esperienza aggiornata è più integrata con macOS e include delle modifiche richieste dall’API. 

Dropbox si impegna attivamente per mantenere l’affidabilità e la qualità della propria esperienza di sincronizzazione con queste modifiche. Se non hai ancora ricevuto il nuovo Dropbox per macOS con supporto per l’API Provider File, è probabile che arriverà entro la fine di maggio 2023.

Note:

  • Lavoriamo direttamente con un piccolo gruppo di utenti macOS con configurazioni complesse per garantire loro una migrazione quanto più fluida possibile. 
  • Alcuni utenti macOS con configurazioni complesse (per esempio coloro che archiviano la loro cartella Dropbox su un’unità esterna), per il momento saranno esclusi dalla migrazione e continueranno con l’esperienza Dropbox attuale.

Sebbene si consigli vivamente di passare all’ultima versione di Dropbox per macOS con supporto per l’API File Provider, la migrazione non è necessaria per continuare a usare Dropbox. Se l’ultimo aggiornamento non è di tuo interesse, potrai continuare ad aprire i file Solo online direttamente dal Finder. 

Ogni volta che è disponibile un aggiornamento a Dropbox per macOS con API File Provider, dovresti visualizzare una notifica sull’icona di Dropbox. Se ti interessa partecipare al programma di accesso anticipato, segui le istruzioni per partecipare:

  • Dropbox Basic, Plus, Professional e Family: attiva le versioni in anteprima e attendi che arrivi la notifica per l’adesione.
  • Dropbox Standard, Advanced, Enterprise ed Education: contatta il team responsabile dell’account o il supporto clienti per maggiori informazioni.
