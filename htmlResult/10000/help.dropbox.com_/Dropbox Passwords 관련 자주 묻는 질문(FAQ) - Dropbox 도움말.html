Dropbox Passwords는 Dropbox Basic, Plus, Family, Professional, Standard, Advanced, Enterprise 사용자에게 제공되는 기능입니다. 협상된 계약이 있거나 HIPAA BAA를 준수하는 팀에 소속되어 있는 경우,

Dropbox Passwords

는 Dropbox가 제공하는 브라우저 확장 프로그램 및 모바일 앱입니다. 이 도움말에는 Dropbox Passwords에 관해 자주 묻는 질문에 대한 답변이 정리되어 있습니다.

Dropbox Passwords에 저장 가능한 로그인 정보와 결제 카드 개수

저장할 수 있는 로그인 정보 및 결제 카드 개수, Dropbox Passwords를 사용할 수 있는 장치 개수는 사용 중이신 요금제에 따라 다릅니다. 사용 중이신 요금제를 확인하세요.

  • Dropbox Basic 사용자는 로그인 정보와 결제 카드 정보를 필요한 만큼 저장할 수 있으며, 최대 장치 3대에서 Dropbox Passwords를 사용할 수 있습니다.
  • Dropbox Plus, Professional, Family, Business 사용자는 로그인 정보와 결제 카드 정보를 무제한으로 저장할 수 있으며, Dropbox Passwords를 사용할 수 있는 장치 대수도 제한되지 않습니다.
    • 참고: Dropbox Standard, Advanced, Enterprise 팀 계정에 자신의 개인용 계정을 연결하면, 해당 개인용 계정에서도 로그인 정보와 결제 카드 정보를 무제한으로 저장하고, 장치를 무제한으로 사용할 수 있습니다.

Dropbox Passwords 사용 장치 추가 방법

새 장치를 연결하려면 앱을 다운로드하거나 브라우저 확장 프로그램을 설치하고 로그인합니다. 연결할 새 장치를 확인하라는 알림이 이미 Dropbox Passwords를 사용 중인 모든 장치에 발송됩니다.

  • 연결된 장치에서 알림을 자동으로 받지 못한 경우, Dropbox Passwords 앱 또는 브라우저 확장 프로그램을 열면 새 장치를 확인하라는 메시지가 표시됩니다.
  • 연결된 장치가 없으면, 앱을 설정할 때 받은 복구 단어를 원래 순서대로 입력하라는 메시지가 표시됩니다. 아래에서 복구 단어에 대해 자세히 알아보세요.
  • Dropbox Basic 사용자이며 이미 장치를 3대 연결한 경우, 장치를 더 많이 연결하려면 요금제를 업그레이드해야 합니다. 연결된 각 브라우저 확장 프로그램 또는 모바일 앱이 장치 한도에 포함됩니다.

장치에서 Dropbox Passwords 연결 해제하기

Dropbox Passwords에서 장치 연결을 해제하려면 안내에 따라 장치에서 원격으로 로그아웃하세요.

자동 완성 켜기

Dropbox Passwords에서 자동 완성을 켜면 웹사이트와 앱에서 사용자 이름과 비밀번호가 자동으로 채워집니다. 아래 순서에 따라 설정하세요.

브라우저 확장 프로그램에서:

  1. Dropbox Passwords 브라우저 확장 프로그램을 엽니다.
  2. 아바타(프로필 사진 또는 이니셜)를 클릭합니다.
  3. 환경설정을 클릭합니다.
  4. [설정]을 클릭합니다.
  5. 웹사이트에 로그인할 때 사용자 이름과 비밀번호를 자동 완성합니다 옆의 체크박스에 체크합니다.
    • Dropbox Passwords가 사용자 이름과 비밀번호를 저장하는 방식을 지정하기 위해 드롭다운 메뉴를 클릭하고 항상, 저장할지 묻기, 저장 안 함 중 원하는 항목을 선택합니다.

모바일 앱에서:

  1. Dropbox Passwords 모바일 앱을 엽니다.
  2. 톱니바퀴 아이콘을 누릅니다.
  3. 자동 완성 켜기를 누릅니다.
  4. 설정으로 이동을 누르거나(Android) 표시되는 안내 메시지를 따릅니다(iPhone/iPad).

자동 잠금 사용 방법

Dropbox Passwords 브라우저 확장 프로그램 또는 모바일 앱이 자동으로 잠기도록 설정할 수 있습니다. Dropbox Passwords 모바일 앱에서는 장치의 현재 기본 보안 기능을 사용해서 장치를 잠금 해제합니다. 이러한 기능으로는 해당 장치 및 설정에 따라 PIN, 암호, 얼굴 인식, 지문 ID 또는 기타 숫자나 생체인식 보안 기능이 있습니다.

모바일 앱에서 자동 잠금을 켜려면 다음과 같이 하세요.

  1. Dropbox Passwords 모바일 앱을 엽니다.
  2. 톱니바퀴 아이콘을 누릅니다.
  3. 자동 잠금 사용 을 켭니다.

5분 동안 활동이 없거나 장치가 잠금 상태이면 앱이 자동으로 잠깁니다.

브라우저 확장 프로그램에서 자동 잠금을 켜려면 다음과 같이 하세요.

  1. Dropbox Passwords 브라우저 확장 프로그램을 엽니다.
  2. 아바타(프로필 사진 또는 이니셜)를 클릭합니다.
  3. 환경설정을 클릭합니다.
  4. 개인정보 보호 탭을 클릭합니다.
  5. 자동 잠금 아래의 암호 추가를 클릭합니다.
  6. 암호를 입력합니다.
  7. 저장을 클릭합니다.
  8. 확인을 클릭하여 자동 잠금을 켭니다.
    • 특정 시간이 지난 후에 앱이 자동으로 잠기도록 설정하려면, 잠금 전 경과 시간 옆의 드롭다운을 클릭하고 30분, 1시간, 2시간, 12시간, 24시간을 선택하거나 안 함을 선택하십시오. 기본적으로, 사용자가 옵션을 선택하지 않은 경우 앱은 12시간이 지나면 자동으로 잠깁니다.
    • Chrome 또는 Microsoft Edge에서 Dropbox Passwords를 사용 중인 경우 컴퓨터 절전 모드 시 잠금을 켜도록 확인란을 선택할 수도 있습니다. 사용자가 선택한 자동 잠금 시간에 관계없이 컴퓨터가 절전 모드 상태로 들어가면 항상 자동으로 잠깁니다.

복구 단어의 용도

Dropbox Passwords를 처음으로 설정할 때 12개의 복구 단어를 저장하여 보관해 놓으라는 메시지가 표시됩니다.

Dropbox Passwords에서는 보안 강화를 위해 이미 연결된 장치에 알림을 보내서 새로 연결하려는 장치를 확인합니다. 이전에 연결해 놓은 장치에 액세스할 수 없는 경우에는 복구 단어를 원래 순서대로 입력하라는 메시지가 표시됩니다. 복구 단어가 없으면 새 장치에 로그인할 수 없으며 Dropbox Passwords 계정에 액세스할 수 없게 됩니다.

복구 단어 확인 및 저장 방법

복구 단어는 Dropbox Passwords가 연결된 장치에서 언제든 확인하고 저장할 수 있습니다.

복구 단어 확인하고 저장하기:

  1. Dropbox Passwords 브라우저 확장 프로그램을 엽니다.
  2. 아바타(프로필 사진 또는 이니셜)를 클릭합니다.
  3. 환경설정을 클릭합니다.
  4. 개인정보 보호 탭을 클릭합니다.
  5. 내 단어 보기 및 저장하기를 클릭합니다.
  6. 단어를 저장하려면 PDF 저장을, 창을 닫으려면 닫기를 클릭합니다.

Dropbox 요금제 변경 시의 Dropbox Passwords 사용

Dropbox 계정을 Basic으로 다운그레이드하더라도 Dropbox Passwords 계정은 계속 이용할 수 있습니다. 그러나 이미 장치를 3대 이상 연결한 경우, 그 외의 장치에서는 Dropbox Passwords를 사용할 수 없습니다.

Dropbox Plus, Professional, Family 요금제 안에서 다른 요금제로 업그레이드하거나 다운그레이드하는 경우 Dropbox Passwords 사용에는 아무런 변화가 없습니다.

Dropbox Passwords 기능이 활성화되지 않은 팀 계정으로 업그레이드하는 경우, 기존에 Dropbox Passwords에 저장해둔 사용자 이름과 비밀번호를 내보낼 수는 있지만 다음과 같은 Dropbox Passwords 기능은 더 이상 사용할 수 없습니다.

  • 사용자 이름 및 비밀번호 보기, 수정, 추가 또는 제거
  • 장치 간 동기화
  • 웹사이트 및 모바일 앱에서 자동 완성
  • Dropbox Passwords 계정에 저장된 모든 정보 삭제
    • 참고: 사용자 이름과 비밀번호는 내보내도 삭제되지 않습니다.

원래 사용하던 Dropbox Plus, Professional, Family 요금제로 계정을 복귀하면 기존에 연결했던 장치나 복구 단어를 사용해 Dropbox Passwords 계정을 재활성화할 수 있습니다. 이렇게 계정이 다시 활성화되면 저장되어 있던 정보가 그대로 나옵니다.

Dropbox Passwords 계정 삭제 방법

Dropbox Passwords 계정을 삭제하려면 Dropbox 계정을 삭제해야 합니다. 두 계정이 연결되어 있기 때문입니다. 두 계정을 모두 삭제하지 않으려면 Dropbox Passwords에 저장해둔 모든 계정 정보(사용자 이름, 비밀번호, 웹사이트 URL, 메모 등)를 전부 삭제하는 방법도 있습니다.

Dropbox Passwords 계정 재설정 방법

Dropbox Passwords 계정을 재설정하면 사용자 이름, 비밀번호, 웹사이트 URL, 메모 등 Dropbox Passwords에 저장한 모든 계정 정보가 모든 장치에서 영구 삭제되며, 연결되었던 장치는 연결 해제됩니다. 연결 해제된 장치에 Dropbox Passwords 앱이 설치되어 있기는 하지만, 다시 연결해서 로그인하려면 해당 장치를 재확인해야 합니다.

브라우저 확장 프로그램에서의 재설정 방법

  1. Dropbox Passwords 브라우저 확장 프로그램을 엽니다.
  2. 아바타(프로필 사진 또는 이니셜)를 클릭합니다.
  3. 환경설정을 클릭합니다.
  4. 계정 탭을 클릭합니다.
  5. 재설정을 클릭합니다.
  6. 계속을 클릭합니다.
  7. 계정을 재설정하기 전에 내보내려면 내보내기를 클릭합니다. 그렇지 않으면 건너뛰기를 클릭합니다.
  8. Dropbox Passwords 패스프레이즈를 입력합니다.
  9. 계속을 클릭합니다.
  10. 재설정을 클릭합니다.

모바일 앱에서 재설정하려면 다음과 같이 하세요.

  1. Dropbox Passwords 모바일 앱을 엽니다.
  2. 톱니바퀴 아이콘을 누릅니다.
  3. 계정 재설정을 누릅니다.
  4. 계정을 전부 제거하려면 재설정을 누릅니다.

Dropbox Passwords 계정을 삭제하려면 Dropbox 계정을 삭제해야 합니다. 두 계정이 동일한 이메일과 비밀번호로 연결되어 있기 때문입니다.
