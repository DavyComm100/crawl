Dropbox Scan est une application de scanner mobile pour iOS qui vous permet de numériser et d'organiser vos documents et images sur votre appareil mobile. Vous pouvez également connecter votre compte Dropbox de sorte à regrouper automatiquement vos documents numérisés dans un dossier appelé Versions numérisées. Pour vous connecter, ouvrez l'application Dropbox Scan, cliquez sur Se connecter à côté du logo Dropbox, puis suivez les instructions.

Remarque : vous ne pouvez pas connecter de compte Dropbox Business dans l'application Dropbox Scan. Tous les utilisateurs, y compris les utilisateurs Dropbox Business, peuvent numériser des documents directement dans leur compte Dropbox avec l'application mobile Dropbox.

Numérisation d'un document

Pour numériser un document :

  1. Ouvrez l'application Dropbox Scan sur votre appareil mobile.
  2. Appuyez sur l'icône représentant un appareil photo en bas à droite de l'écran.
  3. Prenez une photo du contenu à numériser ou sélectionnez une image existante dans la pellicule de votre appareil photo.
  4. Si vous le souhaitez, renommez les pages, apportez-y des modifications et ajoutez-en ou partagez le document numérisé.
    • Pour renommer un document, appuyez sur l'icône représentant un crayon en haut de l'écran.
    • Pour apporter des modifications, appuyez sur l'icône représentant trois glissières sous le document numérisé.
    • Pour ajouter d'autres pages, appuyez sur Ajouter une page.
    • Pour réorganiser les pages du document numérisé, appuyez sur Réorganiser.
    • Pour partager le document numérisé, appuyez sur l'icône de partage (un rectangle avec une flèche vers le haut) en bas à gauche de l'écran.
  5. Appuyez sur OK.

Remarques : 

  • Les documents numérisés comportant plusieurs pages peuvent être enregistrés au format .pdf.
  • À partir du moment où vous avez enregistré un document numérisé, vous ne pouvez pas le modifier ni lui ajouter d'autres pages.
