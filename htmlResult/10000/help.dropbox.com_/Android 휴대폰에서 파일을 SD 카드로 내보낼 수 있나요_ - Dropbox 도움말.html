Android 휴대폰에서 Dropbox의 파일을 SD 카드로 내보내려면 먼저 Dropbox 앱이 올바른 버전인지 확인해야 합니다. Android 4.4 이상 버전을 사용하는 경우, Dropbox 앱 12 이상 버전이 필요합니다.

최신 버전의 Dropbox 앱은 Google Play 스토어에서 다운로드할 수 있습니다.

Dropbox 앱 버전을 확인했으면 이제 파일을 SD 카드로 내보낼 수 있습니다. 방법은 다음과 같습니다.

  1. 내보낼 파일 이름 우측의 빠른 작업 버튼을 누릅니다.
  2. More(더보기)를 누릅니다.
  3. 내보내기를 누릅니다.
  4. 장치에 저장을 누릅니다.
  5. 외장 SD 카드를 선택합니다.

참고: 외장 SD 카드가 보이지 않을 경우, 다음과 같이 하세요.

  1. 장치에 저장을 누른 다음 메뉴 버튼을 누릅니다.
  2. [설정]을 누릅니다.
  3. [Display advanced devices(고급 디바이스 표시)]를 선택합니다.
  4. 이제 내보내기 위치로 외장 SD 카드가 보일 겁니다.

장치에서 파일을 다른 앱으로 내보내면 어떻게 되나요?

파일을 다른 앱으로 내보내는 경우, 선택한 앱에서 해당 파일이 열립니다. 그러면 이 앱이 파일을 처리합니다. 예를 들어, 파일을 장치의 갤러리 앱으로 내보낸 경우, 갤러리에 파일이 표시되나, 실제 저장되는 것은 아닙니다.

파일을 다른 위치에 저장하려면 'SD 카드 저장'을 선택합니다.
