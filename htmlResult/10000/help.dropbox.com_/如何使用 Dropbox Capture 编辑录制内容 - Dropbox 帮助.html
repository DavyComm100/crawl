您可以选择移除视频的特定部分，或者将视频修剪成所需的长度或片段。
注意：

  • 使用 Dropbox Basic、Plus 和 Family 套餐的客户最多只能编辑时长 5 分钟的视频。
  • 使用 Dropbox Professional、Standard、Advanced 和 Enterprise 套餐的客户可以编辑时长不超过 30 分钟的视频。
  • 只有使用 Dropbox Professional、Standard、Advanced 和 Enterprise 套餐的客户可以移除填充词并创建 GIF。
  • 所有录制文件的大小上限为 2GB。

要编辑使用 Dropbox Capture 制作的视频，请执行以下操作：

  1. 通过 capture.dropbox.com 打开视频。
  2. 在右侧边栏中单击编辑视频。
    • 您也可以从任务栏 (Windows) 或菜单栏 (Mac) 中打开 Dropbox Capture 桌面应用，将光标悬停在一个视频上，然后单击剪刀图标进行编辑。
  3. 使用窗口底部的控件选择视频的一部分。
  4. 根据您的操作意向，单击保留选择或删除选择。
  5. 单击保存更改。

保存后，已编辑的视频将取代原始文件。

单击文件名右侧的铅笔图标，即可重命名捕捉内容。

如何移除填充词

如果包含音频的视频录制内容的时长少于 30 分钟，您可以自动修剪视频，以移除音频中的“嗯”和“呃”等词。

操作步骤如下：

  1. 登录 capture.dropbox.com。
  2. 单击您要移除其填充词的录制内容。
  3. 单击编辑。
  4. 单击移除填充词。
    • 注意：只有在视频已转录完毕且至少可以找到 1 个填充词的情况下，此选项才会启用。移除完毕后，您可以在编辑预览中看到，被移除的填充词上会显示黄色线条。
  5. 单击保存更改。

移除填充词可能需要几分钟的时间才能完成，具体取决于视频的长度。

如何创建 GIF

您可以使用 Dropbox Capture 创建 GIF 文件。

注意：录制内容时长不得超过 30 秒。创建 GIF 的过程可能需要几分钟的时间才能完成，具体取决于录制内容的长度。

在 capture.dropbox.com 上

要在 capture.dropbox.com 上创建 GIF，请执行以下操作：

  1. 登录 capture.dropbox.com。
  2. 单击您要转换为 GIF 的录制内容。
  3. 单击共享旁边的向下箭头。
  4. 单击将视频下载为 GIF。
  5. 预览 GIF，然后单击下载。

您的 GIF 将下载到电脑中。Dropbox Capture 中的原始录制内容将不受影响。


在 Dropbox Capture 应用中

您可以在 macOS 上使用 Dropbox Capture 应用来创建 GIF。为此，请执行以下操作：

  1. 打开 Dropbox Capture 桌面应用。
  2. 单击显示更多。
  3. 单击 GIF。
  4. 选择您要捕捉屏幕的哪部分。
    • 单击一次，可录制整个屏幕。
    • 单击并拖动，可录制屏幕上的所需区域。
    • 按下空格键，即可录制特定窗口。
  5. 单击停止图标（带有白色正方形的红色圆圈）。
  6. 您的 GIF 将自动保存到 Dropbox Capture。

适用于 Windows 的 Dropbox Capture 应用目前无法创建 GIF。
