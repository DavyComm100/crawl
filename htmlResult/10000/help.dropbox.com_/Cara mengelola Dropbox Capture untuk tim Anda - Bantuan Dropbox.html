Beberapa jenis admin tertentu

dapat mengaktifkan atau menonaktifkan

Dropbox Capture

bagi tim mereka, serta mengelola pengaturan teks tertutup untuk Dropbox Capture. Anggota tim dapat melihat dan mengomentari rekaman yang dibagikan di

capture.dropbox.com

serta menambahkan rekaman ke koleksi tim.

Cara mengizinkan atau memblokir penggunaan Dropbox Capture bagi tim Anda

Untuk mengizinkan penggunaan Dropbox Capture bagi tim Anda:

  1. Masuk ke dropbox.com menggunakan kredensial admin Anda.
  2. Klik Konsol Admin.
  3. Klik Pengaturan. 
  4. Di bagian Aplikasi, klik Izin aplikasi.
  5. Cari “Dropbox Capture" di bilah pencarian.
  6. Centang kotak di samping Dropbox Capture pada daftar.
  7. Izinkan atau blokir penggunaan Dropbox Capture:
    • Klik tombol Izinkan berwarna biru untuk mengizinkan penggunaan Dropbox Capture.
    • Klik tombol Blokir berwarna putih untuk memblokir penggunaan Dropbox Capture.

Ketahui selanjutnya tentang pengelolaan aplikasi untuk tim Anda.

Cara mengaktifkan atau menonaktifkan teks tertutup Dropbox Capture bagi tim Anda

Untuk mengaktifkan atau menonaktifkan teks bagi tim Anda:

  1. Masuk ke dropbox.com menggunakan kredensial admin Anda.
  2. Klik Konsol Admin.
  3. Klik Pengaturan. 
  4. Klik Capture. 
  5. Aktifkan atau nonaktifkan Teks Tertutup . 
  6. Klik Save.

Ketahui selanjutnya tentang penggunaan teks tertutup pada Dropbox Capture.

Cara menggunakan koleksi dengan tim Anda

Tim Saya adalah ruang bersama bagi tim Dropbox. Anda bisa membuat folder tempat anggota tim dapat berkolaborasi mengelola capture, menambahkan komentar, dan menerima pemberitahuan otomatis mengenai setiap perubahan. 

Setelah Anda membuat koleksi tim, cukup tambahkan capture dan semua orang yang terkait dengan koleksi tersebut akan otomatis menerima pemberitahuan, jika tim Anda berisi 30 orang atau kurang. 

Untuk membuat koleksi baru
Untuk membuat koleksi baru:

  1. Masuk ke Dropbox Capture. 
  2. Klik Buat Koleksi di samping Koleksi tim. 
  3. Masukkan nama untuk menjadi judul koleksi Anda dan klik tanda centang. 

Untuk menghapus koleksi
Untuk menghapus koleksi:

  1. Masuk ke Dropbox Capture. 
  2. Gulir ke koleksi yang ingin Anda hapus. 
  3. Arahkan kursor ke nama koleksi dan klik “…”.
  4. Klik Hapus. 
  5. Klik Hapus sekali lagi untuk mengonfirmasi.
