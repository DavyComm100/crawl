Dropbox 모바일 앱을 사용하면 휴대폰 또는 태블릿에서 파일을 만들어 Dropbox 계정에 바로 업로드할 수 있습니다. 업데이트된 파일은 Dropbox의 파일에 액세스한 모든 곳에서 동기화됩니다.

Dropbox를 아직 사용하고 있지 않으신가요? Dropbox로 어떻게 파일을 쉽게 동기화하는지 살펴보세요.

다른 모바일 앱의 콘텐츠를 업로드하는 방법

사진, 메시지 등 다른 모바일 앱의 콘텐츠를 Dropbox 계정에 업로드할 수 있습니다. 방법은 다음과 같습니다.

  1. 현재 콘텐츠가 저장되어 있는 앱을 엽니다.
  2. Dropbox 계정에 업로드하려는 항목을 선택합니다.
  3. 앱의 공유 버튼 또는 아이콘을 클릭합니다.
  4. Dropbox 모바일 앱을 선택합니다.
    • 참고: Dropbox 모바일 앱에서 Dropbox 계정에 로그인해야 합니다.
  5. Dropbox 저장 위치를 선택하고 콘텐츠를 저장합니다.

사진과 동영상을 업로드하는 방법

카메라 업로드를 사용 설정하지 않은 경우에는 사진을 Dropbox 계정에 직접 올릴 수 있습니다. 방법은 다음과 같습니다.

  1. Dropbox 모바일 앱을 엽니다. 
  2. 사진을 저장하려는 Dropbox 폴더를 엽니다.
  3. 화면 하단에 '+' (더하기 기호)를 누릅니다.
  4. 사진 업로드를 선택합니다.
  5. 업로드하려는 사진을 누릅니다.
  6. 폴더 아이콘(Android의 경우) 또는 다음(iOS의 경우)을 누른 다음 파일을 올릴 위치를 선택합니다.
  7. 위치 설정을 누릅니다.
  8. 업로드를 누릅니다.

참고: Windows 장치용 Dropbox 앱은 현재 동영상 업로드를 지원하지 않습니다. 이는 운영 체제의 제한으로 인한 것입니다.

오디오 파일을 업로드하는 방법

iOS 사용자는 Dropbox 모바일 앱을 통해 오디오를 녹음하여 Dropbox 계정에 바로 저장할 수 있습니다. 방법은 다음과 같습니다.

  1. Dropbox 앱을 엽니다.
  2. 화면 하단에서 +(더하기) 아이콘을 누릅니다.
  3. 오디오 녹음을 누릅니다.
  4. 빨간색 원을 누르면 녹음이 시작되고 빨간색 사각형을 누르면 녹음이 중지됩니다.
  5. 오디오 파일을 저장하려는 Dropbox 폴더를 선택합니다.
  6. 업로드를 누릅니다.

참고: Windows S 모드용 Dropbox 모바일 앱(Dropbox Lite)과 Android 장치용 Dropbox 모바일 앱에서는 현재 오디오 캡처를 지원하지 않습니다.

다른 형식의 파일을 업로드하는 방법

  1. Dropbox 모바일 앱을 엽니다.
  2. 파일을 저장하려는 Dropbox 폴더를 엽니다.
  3. 화면 하단에 '+' (더하기 기호)를 누릅니다.
  4. 파일 업로드(Android의 경우) 또는 파일 만들기 또는 업로드(iPhone/iPad의 경우)를 누릅니다.
  5. 업로드(Android의 경우) 또는 파일 업로드(iPhone/iPad의 경우)를 누릅니다.

장치에 SD 카드가 장착되어 있는 경우, SD 카드에 있는 파일을 Dropbox로 업로드할 수 있습니다. 방법은 다음과 같습니다.

  1. Dropbox 모바일 앱을 엽니다.
  2. 사진을 저장하려는 Dropbox 폴더를 엽니다.
  3. 화면 하단에 '+' (더하기 기호)를 누릅니다.
  4. 파일 업로드를 선택합니다.
  5. SD 카드를 찾은 다음 업로드할 파일을 누릅니다.
  6. 업로드(일부 장치의 경우 열기)를 누릅니다.

참고: 업로드할 파일을 찾는데 파일이 보관된 SD 카드가 보이지 않는 경우에는 화면 위쪽에 있는 메뉴 버튼을 눌러 설정 아래에 고급 장치 보기가 선택되어 있는지 확인합니다.

iPhone/iPad 사용자는 다음과 같은 방법으로도 다른 형식의 파일을 업로드할 수 있습니다.

  • iPhone/iPad의 다음에서 열기 메뉴(다른 앱의 어느 위치에서든 표시되는 경우)
  • iOS 8 이상의 앱 확장 프로그램
