Si vous souhaitez exporter des fichiers à partir de Dropbox vers une carte SD sur votre téléphone Android, vous devez d'abord vérifier que vous disposez de la version appropriée de l'application Dropbox. Si vous utilisez Android 4.4 ou une version ultérieure, vous aurez besoin de la version 12 (ou ultérieure) de l'application Dropbox.

Obtenir la version la plus récente de l'application Dropbox à partir du Google Play Store

Une fois que vous avez vérifié que vous possédiez la bonne version de l'application Dropbox, procédez comme suit :

  1. Appuyez sur le bouton d'action rapide à droite du nom du fichier à exporter.
  2. Appuyez sur Plus.
  3. Appuyez sur Exporter.
  4. Appuyez sur Enregistrer sur l'appareil.
  5. Sélectionnez votre carte SD externe.

Remarque : si vous ne voyez pas votre carte SD externe dans la liste, procédez comme suit :

  1. Après avoir appuyé sur le bouton Enregistrer sur l'appareil, appuyez sur le bouton Menu.
  2. Appuyez sur Paramètres.
  3. Sélectionnez Afficher les appareils avancés.
  4. Votre carte SD externe doit alors s'afficher en tant qu'emplacement d'exportation.

Que se passe-t-il lorsque j'exporte un fichier vers une autre application sur mon appareil ?

Lorsque vous exportez un fichier vers une autre application, celui-ci s'ouvre dans l'application sélectionnée. Cette dernière prend alors en charge le fichier. Par exemple, si vous exportez un fichier vers l'application Galerie de votre appareil, cette dernière affiche le fichier, mais ne le stocke pas.

Pour stocker un fichier à un autre emplacement, choisissez l'option "Enregistrer sur la carte SD".
