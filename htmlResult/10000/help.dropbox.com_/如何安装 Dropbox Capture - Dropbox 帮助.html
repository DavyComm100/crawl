Dropbox Capture

是一个供用户创建屏幕截图、录屏、录音和视频消息并使用这些内容交流的工具。

如何安装 Dropbox Capture

要安装 Dropbox Capture，请执行以下操作：

  1. 登录 dropbox.com。
  2. 前往 dropbox.com/capture。
  3. 单击下载。
  4. 在 Capture 下载完成后，启动该文件即可开始安装。
  5. 完成安装流程。
  6. Capture 安装完毕后，单击任务栏 (Windows) 或菜单栏 (Mac) 中的 Dropbox Capture 图标（钻石形镜头），即可打开该应用。

第一次打开 Dropbox Capture 时，您需要使用您的 Dropbox 用户名和密码登录。如果您已登录 Dropbox 桌面应用，您将通过此帐户自动登录 Capture。

打开后，您可以单击右上角的“⁝”（垂直省略号），访问更新、首选项、产品更新、帮助等。

注意：如果您在 Windows 上遇到 Dropbox Capture 安装问题，我们建议检查系统更新，然后重置电脑并尝试重新安装。或者，从 Microsoft Store 安装 Dropbox Capture。

如何更新 Dropbox Capture

Windows 用户可以关闭并重新启动 Capture 以进行更新，或者单击此处以重新安装应用。

Mac 用户可以单击应用右上角的“⁝”（垂直省略号），选择检查更新。

如何更改 Dropbox Capture 的主题

默认情况下，Dropbox Capture 的主题将自动与您的电脑相匹配。要将主题更改为浅色模式或深色模式，请执行以下操作：

  1. 打开 Dropbox Capture 桌面应用。
  2. 单击屏幕右上角的“⁝”（垂直省略号）。
  3. 选择首选项。
  4. 单击主题旁边的下拉菜单，选择一个选项。

如何将 Dropbox Capture 添加到 Windows 任务栏

要将 Capture 添加到 Windows 任务栏以快速访问，请执行以下操作：

  1. 打开 Dropbox Capture。
  2. 单击屏幕右上角的“⁝”（垂直省略号）。
  3. 选择首选项。
  4. 将在任务栏中显示切换至开启。

如何将 Dropbox Capture 添加到 macOS 程序坞

要将 Capture 添加到 macOS 程序坞以快速访问，请执行以下操作：

  1. 打开 Dropbox Capture。
  2. 单击屏幕右上角的“⁝”（垂直省略号）。
  3. 选择首选项。
  4. 将在程序坞中显示切换至开启。

如何为您创建的视频启用或禁用字幕

您可以在使用 Dropbox Capture 创建的任何视频中添加自动生成的隐藏式字幕。隐藏式字幕使您的内容更容易传达给观众，包括那些有听力障碍或无法播放音频的观众。

启用或禁用字幕时，您将为所有 Dropbox Capture 视频启用/禁用字幕。要为视频启用/禁用字幕，请执行以下操作：

  1. 打开 Dropbox Capture 桌面应用。
  2. 单击右上方的“⁝”（垂直省略号）。
  3. 单击首选项。
  4. 将生成隐藏式字幕切换至开启/关闭。

注意：如果您使用的是 Dropbox 团队帐户，但没有看到生成隐藏式字幕设置，则说明您的管理员已禁用字幕。

如何开启或关闭您观看的视频字幕

观看视频时，单击视频播放器右下角的 CC，即可开启或关闭字幕。

注意：要查看字幕，视频创建者必须在录制时开启生成隐藏式字幕。
