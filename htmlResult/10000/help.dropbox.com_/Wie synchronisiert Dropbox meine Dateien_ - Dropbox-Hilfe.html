Dieser Artikel bietet einen Überblick über die Synchronisierungsfunktionen von Dropbox. Hier erfahren Sie, wie Sie Ihre Dateien in Dropbox speichern und zwischen Ihren Geräten synchronisieren können. 

Wenn Sie Dateien in Dropbox speichern, sind Ihre Dateien überall verfügbar, wo Sie Dropbox verwenden. Dazu gehören:

  • Dropbox.com in einem Webbrowser, auf Ihrem Computer oder Smartphone.
  • Die Dropbox-Desktop-App auf Ihrem Computer.
  • Die mobile Dropbox-App auf Ihrem Smartphone oder Tablet.

Ihre Dateien werden überall aktualisiert, wo Sie Dropbox verwenden. Wenn Sie also eine Datei an einer Stelle hinzufügen oder ändern, wird die Datei überall automatisch aktualisiert.

Nutzen Sie Dropbox bisher noch nicht? Dann entdecken Sie jetzt, wie leicht Dropbox das Synchronisieren von Dateien macht.

Die Synchronisierung Ihrer Dateien mit Dropbox einrichten

Mit einem kostenlosen Dropbox Basic-Abo können Sie alle Dateien auf all Ihren Geräten synchronisieren. Sie erhalten bis zu 2 GB Speicherplatz und haben keine zeitliche Begrenzung.

Sie können jederzeit auf ein zahlungspflichtiges Abo mit mehr Speicherplatz oder zusätzlichen Features upgraden. Weitere Details zu unseren Abos finden Sie hier.

Und so geht's:

  1. Erstellen Sie ein Konto auf dropbox.com/register.
  2. Laden Sie die Dropbox-Desktop-App für Ihren Computer und die Dropbox-App für Mobilgeräte für Ihr Smartphone oder Tablet herunter.
  3. Laden Sie Dateien in Ihr Dropbox-Konto hoch, wählen Sie Dateien zum Synchronisieren von Ihrem Computer oder sichern Sie einen Computer oder eine externe Festplatte.

FAQ 

Kann ich mit der Dropbox-Synchronisierung meine Dateien auf einen neuen Computer verschieben?

Nach den oben genannten Schritten werden Ihre Dateien mit Dropbox synchronisiert. Sie müssen diese also nicht mehr von Hand zwischen Computern oder Geräten verschieben oder übertragen. Über dropbox.com oder die Desktop-App und die mobilen Apps von Dropbox können Sie von jedem Gerät aus auf Ihre Dateien zugreifen. 

Wenn Sie Ihre Dateien auf der Festplatte eines Computers (oder auf mehreren Computern) und online in Ihrem Dropbox-Konto speichern möchten, laden Sie dazu die Dropbox-Desktop-App herunter. Wählen Sie bei Aufforderung anstelle von nur online die Option offline verfügbar.

Hinweis: Die LAN-Synchronisierung wird derzeit nicht auf Computern unterstützt, auf denen Dropbox für macOS mit Apple Dateianbieter ausgeführt wird.

Wodurch unterscheiden sich Dateien, die „offline verfügbar“ oder „nur online“ sind?

Als offline verfügbare Dateien (Standardeinstellung) bezeichnen wir Dateien und Ordner, die verfügbar sind, unabhängig davon, ob Sie mit dem Internet verbunden sind oder nicht, und die daher sowohl auf der Festplatte Ihres Computers als auch in Ihrem Dropbox-Konto Speicherplatz belegen. Wenn Sie Ihre Dateien auf „nur online“ setzen, sehen Sie diese zwar immer noch im Dropbox-Ordner auf Ihrem Computer, aber Sie können sie nur öffnen, wenn Sie mit dem Internet verbunden sind. 

Hinweis: Sobald Sie diese „nur online“ verfügbaren Dateien aus dem Dropbox-Ordner auf Ihrem Computer öffnen, werden sie automatisch auf Ihren Computer heruntergeladen.

Was ist der Unterschied zwischen dem Synchronisieren von Dateien und dem Sichern von Dateien?

Durch das Sichern von Dateien von Ihrem Computer oder externen Laufwerk in Dropbox wird eine Sicherungskopie erstellt, für den Fall, dass etwas schief geht. Gesicherte Dateien sind nicht zum Freigeben gedacht und Änderungen, die Sie an gesicherten Dateien in Dropbox vornehmen, werden auf Ihrem Computer oder externen Laufwerk nicht aktualisiert.

Durch das Synchronisieren von Dateien mit Dropbox werden diese Dateien zum Bearbeiten und Freigeben verfügbar, wo immer Sie Dropbox verwenden. Wann immer Sie eine Änderung an diesen Dateien vornehmen, werden diese Änderungen überall in Dropbox synchronisiert.
