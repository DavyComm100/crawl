Dropbox Capture

ist ein Tool, mit dem Nutzer Screenshots, Bildschirm- und Audioaufzeichnungen sowie Videonachrichten erstellen und für Ihre Kommunikation nutzen können.

Dropbox Capture installieren

So installieren Sie Dropbox Capture:

  1. Melden Sie sich bei dropbox.com an.
  2. Gehen Sie auf dropbox.com/capture.
  3. Klicken Sie auf Herunterladen.
  4. Nachdem Capture heruntergeladen wurde, starten Sie die Datei, um mit der Installation zu beginnen.
  5. Schließen Sie den Installationsvorgang ab. 
  6. Sobald Capture installiert wurde, klicken Sie auf das Dropbox Capture-Symbol (rautenförmiges Objektiv) in der Taskleiste (Windows) bzw. Menüleiste (Mac), um die App zu öffnen.

Wenn Sie Dropbox Capture zum ersten Mal öffnen, werden Sie aufgefordert, sich mit Ihrem Dropbox-Nutzernamen und -Kennwort anzumelden. Wenn Sie bereits bei der Dropbox-Desktop-App angemeldet sind, werden Sie automatisch unter diesem Konto bei Capture angemeldet.

Klicken Sie nach dem Öffnen oben rechts auf die drei vertikalen Punkte („⁝“), um auf Updates, Einstellungen, Produktaktualisierungen, Hilfe und mehr zuzugreifen.

Hinweis: Wenn Sie Probleme bei der Installation von Dropbox Capture unter Windows haben, empfehlen wir, nach System-Updates zu suchen, Ihren Computer neu zu starten und die Installation noch einmal auszuführen. Oder installieren Sie Dropbox Capture aus dem Microsoft Store.

Dropbox Capture aktualisieren

Unter Windows können Sie Capture beenden und neu starten, um ein Update durchzuführen, oder hier klicken, um die App neu zu installieren.

Unter Mac können Sie oben rechts in der App auf die drei vertikalen Punkte („⁝“) klicken und Updates suchen auswählen.

Tastenkombinationen für Dropbox Capture anpassen

So greifen Sie auf Tastenkombinationen zu oder bearbeiten sie:

  1. Öffnen Sie die Dropbox Capture-Desktop-App.
  2. Klicken Sie rechts oben auf dem Bildschirm auf die drei vertikalen Punkte („⁝“).
  3. Wählen Sie Einstellungen aus.
  4. Klicken Sie oben im Menü auf Tastenkombinationen.

Das Design von Dropbox Capture ändern

Standardmäßig passt sich das Design von Dropbox Capture automatisch Ihrem Computer an. So ändern Sie das Design in Hell oder Dunkel:

  1. Öffnen Sie die Dropbox Capture-Desktop-App.
  2. Klicken Sie rechts oben auf dem Bildschirm auf die drei vertikalen Punkte („⁝“).
  3. Wählen Sie Einstellungen aus.
  4. Klicken Sie auf das Drop-down-Menü neben Design und wählen Sie eine Option.

Untertitel für Capture-Videos aktivieren oder deaktivieren

Sie können automatisch generierte Untertitel zu allen Videos hinzufügen, die Sie mit Dropbox Capture erstellen. Untertitel machen Ihre Inhalte besser zugänglich, auch für Personen mit beeinträchtigtem Hörvermögen oder diejenigen, die keinen Ton abspielen können. 

Wenn Sie Untertitel aktivieren oder deaktivieren, aktivieren bzw. deaktivieren Sie die Untertitel für all Ihre Dropbox Capture-Videos. So aktivieren bzw. deaktivieren Sie Untertitel für Ihre Videos:

  1. Öffnen Sie die Dropbox Capture-Desktop-App.
  2. Klicken Sie rechts oben auf die drei vertikalen Punkte („⁝“).
  3. Klicken Sie auf Einstellungen. 
  4. Schalten Sie die Option Untertitel erstellen ein bzw. aus.

Hinweis: Wenn Sie ein Dropbox-Teamkonto haben und die Option Untertitel erstellen nicht sehen, hat Ihr Admin die Untertitel möglicherweise deaktiviert.

Untertitel für Videos, die Sie ansehen, aktivieren oder deaktivieren

Wenn Sie ein Video ansehen, klicken Sie auf CC rechts unten im Videoplayer, um die Untertitel zu aktivieren oder zu deaktivieren.

Hinweis: Damit Untertitel angezeigt werden können, muss die Person, die das Video erstellt hat, zum Aufnahmezeitpunkt die Option Untertitel erstellen aktiviert haben.
